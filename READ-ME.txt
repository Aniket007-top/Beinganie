Current status:
The project has been migrated into a Next.js + Tailwind CSS application inside this folder.
The active application code is in the `app`, `components`, and `lib` directories.

Main implemented features:
- Professional homepage with hero section, search UI, dark/light toggle, and category navigation
- Dedicated subject routes:
  - `/ai`
  - `/iot`
  - `/robotics`
- Dedicated level routes:
  - `/ai/beginner`, `/ai/intermediate`, `/ai/advanced`
  - `/iot/beginner`, `/iot/intermediate`, `/iot/advanced`
  - `/robotics/beginner`, `/robotics/intermediate`, `/robotics/advanced`
- Reusable curriculum layout and level page components
- Arcade Lab features:
  - AI Lab: Train the Model
  - IoT Lab: Smart Home Architect
  - Robotics Lab: Code the Rover
- Local progress behavior using browser localStorage
- Resource hub sections and AI notes link
